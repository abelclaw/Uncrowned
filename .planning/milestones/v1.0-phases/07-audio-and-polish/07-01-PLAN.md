```typescript
export interface RoomAudioData {
    /** Phaser audio key for background music. Rooms sharing the same key skip re-triggering. */
    music?: string;
    /** Ambient loop tracks to layer for atmosphere. */
    ambient?: Array<{ key: string; volume?: number }>;
}
```
Add `audio?: RoomAudioData;` field to the existing `RoomData` interface, after the `npcs` field.

**AudioManager singleton (new file):**
- Import `EventBus` from `../EventBus` and `Phaser` from `phaser`.
- Private constructor, `static getInstance()` pattern (matching GameState singleton).
- Instance state: `scene: Phaser.Scene`, `currentMusic: Phaser.Sound.WebAudioSound | null`, `currentMusicKey: string | null`, `ambientSounds: Map<string, Phaser.Sound.WebAudioSound>`, volume levels (`sfxVolume: 1.0`, `musicVolume: 0.5`, `ambientVolume: 0.4`), `sfxEventMap: Record<string, string>` (loaded from audio-registry.json), `initialized: boolean`.
- `init(scene: Phaser.Scene): void` -- stores scene reference, loads sfxEventMap from `scene.cache.json.get('audio-registry')`, calls `registerEventListeners()`. Sets `initialized = true`. Handles audio unlock: if `scene.sound.locked`, listen for `Phaser.Sound.Events.UNLOCKED`.
- `registerEventListeners(): void` -- iterates `sfxEventMap` entries and for each `[eventName, sfxKey]` calls `EventBus.on(eventName, () => this.playSfx(sfxKey))`. Store handler references in a `private eventHandlers: Map<string, Function>` for cleanup.
- `playSfx(key: string): void` -- guard on `!this.initialized || this.scene.sound.locked`, then `this.scene.sound.play(key, { volume: this.sfxVolume })`.
- `playMusic(key: string, fadeDuration: number = 1000): void` -- if `this.currentMusicKey === key` return (same track). Create new sound via `this.scene.sound.add(key, { loop: true, volume: 0 })` cast to `Phaser.Sound.WebAudioSound`, call `.play()`. Fade in via `this.scene.tweens.add({ targets: newMusic, volume: this.musicVolume, duration: fadeDuration })`. If `this.currentMusic?.isPlaying`, fade out old track with `onComplete` calling `oldMusic.stop(); oldMusic.destroy()`. Update `currentMusic` and `currentMusicKey`.
- `setAmbient(keys: Array<{ key: string; volume?: number }>): void` -- fade out and destroy all existing ambient sounds (iterate `ambientSounds` map, tween volume to 0 over 500ms, `onComplete` stop+destroy), clear map. For each new key, create looping sound at volume 0, play, tween to `(amb.volume ?? 1.0) * this.ambientVolume` over 800ms, store in map.
- `stopAll(): void` -- stop and destroy currentMusic (set null), stop and destroy all ambient sounds (clear map), reset `currentMusicKey`.
- `onRoomEnter(roomData: { audio?: RoomAudioData }): void` -- convenience method: if `roomData.audio?.music`, call `playMusic(roomData.audio.music)`. If `roomData.audio?.ambient`, call `setAmbient(roomData.audio.ambient)`. If no ambient defined, call `setAmbient([])` to clear.
- `cleanup(): void` -- removes all EventBus listeners (iterate `eventHandlers`, `EventBus.off` each), clears eventHandlers map. Does NOT stop audio (audio persists across scene restarts). Sets `initialized = false`.

**Important patterns to follow:**
- Use `as Phaser.Sound.WebAudioSound` cast on `scene.sound.add()` return (Phaser returns `BaseSound`).
- Do NOT use `scene.sound.stopAll()` -- it kills everything globally. Stop specific references.
- Scene-level tweens are fine since AudioManager.init() rebinds to the new scene on each RoomScene.create().

**2. Generate placeholder audio files** using Python with the `wave` module (standard library, no dependencies). Create a script `scripts/generate-placeholder-audio.py` that generates short sine-wave-based audio files:

- SFX files (short, 0.1-0.5 seconds):
  - `sfx/item-pickup.mp3` -- rising two-tone (300Hz -> 600Hz, 0.2s)
  - `sfx/door-transition.mp3` -- swoosh-like descending tone (500Hz -> 200Hz, 0.3s)
  - `sfx/death-sting.mp3` -- dramatic descending chord (400Hz -> 100Hz, 0.5s)
  - `sfx/dialogue-start.mp3` -- soft blip (800Hz, 0.1s)
  - `sfx/command-blip.mp3` -- quick click tone (1000Hz, 0.05s)

- Music files (longer loops, 3-5 seconds, will be set to loop):
  - `music/forest.mp3` -- gentle alternating tones (C4-E4-G4, 4s)
  - `music/cave.mp3` -- low droning tone (C3 + E3, 4s)
  - `music/village.mp3` -- warm alternating tones (F4-A4-C5, 4s)
  - `music/menu.mp3` -- simple melody (C4-E4-G4-C5, 3s)

- Ambient files (medium duration, 2-3 seconds, will loop):
  - `ambient/forest-birds.mp3` -- high-frequency chirps (2000Hz bursts, 3s)
  - `ambient/cave-drips.mp3` -- periodic low plops (200Hz bursts, 3s)
  - `ambient/wind-light.mp3` -- white noise approximation (low amplitude noise, 2s)

Since Python's `wave` module only generates WAV, generate WAV files first then convert with `ffmpeg` if available, or use WAV directly (Phaser supports WAV loading). If ffmpeg is not available, save as .wav and update file extensions accordingly.
