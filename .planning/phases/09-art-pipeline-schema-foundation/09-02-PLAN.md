  <name>Task 2: Art generation build script</name>
  <files>

  <action>
    Create `scripts/generate-art.ts` -- a Node.js/TypeScript build script that generates images via ComfyUI REST API and post-processes with sharp.

    **Script structure:**

    1. **Configuration constants:**
       - `COMFYUI_URL = 'http://127.0.0.1:8188'`
       - Load style-guide.json, art-manifest.json, comfyui-workflow.json from `scripts/` dir
       - `PROMPT_NODE_ID` and `SEED_NODE_ID` constants matching the workflow node IDs

    2. **Core functions:**

       `async function checkComfyUI(): Promise<boolean>` -- GET request to COMFYUI_URL, returns true if reachable.

       `async function generateImage(prompt: string, seed: number): Promise<Buffer>`:
       - Deep-clone the workflow template
       - Inject full prompt (style prefix + room prompt + style suffix) into PROMPT_NODE_ID's inputs.text
       - Inject seed into SEED_NODE_ID's inputs.noise_seed
       - POST to `/prompt` with `{ prompt: workflow }`
       - Poll GET `/history/{prompt_id}` every 3 seconds until complete (timeout after 10 minutes)
       - Extract output image filename from history response
       - GET `/view?filename={filename}&type=output` to download PNG
       - Return as Buffer

       `async function processBackground(input: Buffer, outputPath: string, width: number, height: number): Promise<void>`:
       - Use sharp to resize to exact dimensions with `fit: 'cover'`
       - Save as PNG with compressionLevel 9
       - Create output directory if needed (mkdir -p)

       `async function processSprite(input: Buffer, outputPath: string, width: number, height: number): Promise<void>`:
       - Use sharp with `ensureAlpha()` and `fit: 'contain'` with transparent background `{ r: 0, g: 0, b: 0, alpha: 0 }`
       - Save as PNG

    3. **CLI interface:**
       - Parse args: `--type` (backgrounds|shared|sprites|all), `--room` (specific room ID), `--dry-run` (print prompts without generating)
       - `--type backgrounds` generates all 36 room backgrounds
       - `--type shared` generates 6 shared parallax layers
       - `--type all` generates everything
       - `--room forest_clearing` generates just one room (useful for testing)
       - `--dry-run` prints each prompt + seed + output path without calling ComfyUI

    4. **Main execution flow:**
       - Check ComfyUI availability, exit with helpful message if not running
       - Read manifest entries based on --type/--room filters
       - For each entry: check if output file already exists (skip if so, unless --force flag)
       - Generate image via ComfyUI
       - Post-process with sharp
       - Log progress: `[3/36] Generated forest_clearing.png (4.2s)`
       - Report summary at end: N generated, N skipped (already exist), N failed

    5. **Error handling:**
       - ComfyUI connection failure: print instructions to start ComfyUI
       - Generation timeout: log error, continue to next image
       - sharp processing failure: log error, save raw image as fallback
       - All errors are non-fatal for batch processing (skip failed, continue)

    **Run with:** `npx tsx scripts/generate-art.ts --type backgrounds --room forest_clearing`

    NOTE: This script is for offline use only. It requires ComfyUI running locally. It is NOT imported by any game code.
