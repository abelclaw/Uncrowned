3. Implement `static wipeToRoom(scene, roomId, spawnPoint, direction, duration)`:
   - Create a black `Phaser.GameObjects.Rectangle(0, 0, 0, 540)` at depth 1000, setScrollFactor(0), setOrigin(0, 0).
   - Tween its width from 0 to 960 over `duration` ms. Direction determines starting edge: `wipe-left` starts from right edge (x=960, tweens width growing left), `wipe-right` starts from left edge (x=0, tweens width growing right).
   - On complete: `scene.scene.start('RoomScene', { roomId, spawnPoint, transitionFrom: direction })`.
   - For simplicity: set the rect x to 0 for wipe-right (grows right to left visually), or x to 960 and use negative tween for wipe-left. Actually: use a simpler approach -- tween a full-screen black rect's alpha or position. Best approach:
     - `wipe-right`: rect starts at x=0, width=0; tween width to 960. Wipes from left to right.
     - `wipe-left`: rect starts at x=960, width=0, origin(1,0); tween width to 960. Wipes from right to left.

4. Implement `static pixelateToRoom(scene, roomId, spawnPoint, duration)`:
   - Use Phaser's PostFX pixelate: `const fx = scene.cameras.main.postFX.addPixelate(1)` (starts at 1 = no pixelation).
   - Tween the pixelate amount from 1 to 20 over `duration * 0.5` ms (pixelation grows).
   - Then fade to black over `duration * 0.5` ms using `scene.cameras.main.fadeOut()`.
   - On FADE_OUT_COMPLETE: remove pixelate effect via `scene.cameras.main.postFX.remove(fx)`, then start new scene.
   - Pass `{ roomId, spawnPoint, transitionFrom: 'pixelate' }`.

5. Implement `static irisToRoom(scene, roomId, spawnPoint, duration)`:
   - Create a `Phaser.GameObjects.Graphics()` at depth 1000, setScrollFactor(0).
   - Use a mask approach: draw a filled black rect covering the whole screen, then cut a circle out. Tween the circle radius from large to 0.
   - Simpler approach using a filled overlay: Create graphics, fill entire screen black. Create a circle-shaped mask centered on the player. Actually -- simplest Phaser 3 approach:
     - Use `scene.cameras.main.fadeOut(duration, 0, 0, 0)` as the base.
     - Add a vignette PostFX that ramps up: `const fx = scene.cameras.main.postFX.addVignette(0.5, 0.5, 0.3)`. Tween the vignette radius from 1.0 to 0.0 over `duration` ms.
     - On complete: remove vignette, start scene.
   - Actually, for a true iris effect, use a circle mask:
     - Create a `Graphics` object, draw a white filled circle at screen center.
     - Create a `BitmapMask` from it. Apply to camera: `scene.cameras.main.setMask(mask)`.
     - Tween the circle radius from `Math.max(GAME_WIDTH, GAME_HEIGHT)` down to 0 over `duration` ms by clearing and redrawing the graphics each frame.
     - This is heavy. Better approach: use the vignette PostFX trick which is visually similar and much simpler. Tween vignette from radius 1.0 (no vignette) to radius 0.0 (complete blackout) over duration. This creates an iris-close effect.
     - `const fx = scene.cameras.main.postFX.addVignette(0.5, 0.5, 0.0, 1.0)` -- starts with no darkening, radius 1.0.
     - Tween: `scene.tweens.add({ targets: fx, strength: 1.0, duration, onComplete: ... })` -- ramps strength from 0 to 1 creating iris close.
     - On complete: `scene.cameras.main.postFX.remove(fx)`, then `scene.scene.start(...)`.
   - Pass `{ roomId, spawnPoint, transitionFrom: 'iris' }`.

```typescript
  cavern_west_wing: 2, cavern_balcony: 2, underground_pool: 2,
  underground_river: 2, echo_chamber: 2, crystal_chamber: 2,
  forge_chamber: 2, guardian_chamber: 2, dungeon: 2,
  // Act 3: return + climax
  castle_courtyard_act3: 3, castle_hallway: 3, royal_archive: 3,
  mirror_hall: 3, clock_tower: 3, treasury: 3,
  waiting_room: 3, filing_room: 3, throne_room_act3: 3,
};

static isActChange(fromRoom: string, toRoom: string): boolean {
  const fromAct = SceneTransition.ACT_ROOMS[fromRoom];
  const toAct = SceneTransition.ACT_ROOMS[toRoom];
  return fromAct !== undefined && toAct !== undefined && fromAct !== toAct;
}
```
