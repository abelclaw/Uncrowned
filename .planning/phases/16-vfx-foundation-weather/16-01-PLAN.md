     export interface RoomEffectsData {
       weather?: {

2. In `src/game/systems/EffectsManager.ts`:
   - Create a singleton class following the AudioManager pattern exactly:
     - `private static instance`, `private constructor()`, `static getInstance()`
     - `private scene: Phaser.Scene`
     - `init(scene: Phaser.Scene)` binds to current scene
     - `cleanup()` destroys all active emitters, called from RoomScene shutdown
   - Store active emitters in `private weatherEmitter: Phaser.GameObjects.Particles.ParticleEmitter | null`
   - Store ambient emitters in `private ambientEmitters: Phaser.GameObjects.Particles.ParticleEmitter[]`
   - Add `onRoomEnter(roomData: { effects?: RoomEffectsData })` method that:
     - Calls `clearAll()` to destroy existing emitters
     - If `roomData.effects?.weather` exists, calls `createWeatherEmitter(type, intensity)`
     - If `roomData.effects?.ambient` exists, calls `createAmbientEmitter(type, intensity)` for each
   - Add `clearAll()` private method that destroys all emitters and nulls references

3. Weather preset method `private createWeatherEmitter(type: WeatherType, intensity: number)`:
   - Use `this.scene.add.particles(0, 0, '__DEFAULT')` for procedural particles (Phaser 3.60+ uses texture key; use a 2x2 white pixel created via `this.scene.make.graphics({}).generateTexture('particle-white', 2, 2)` if 'particle-white' texture doesn't exist yet)
   - Create the texture once in `init()`: check `this.scene.textures.exists('particle-white')`, if not, create a 2x2 white filled graphics, call `generateTexture('particle-white', 2, 2)`, destroy the graphics
   - Set emitter depth to 80 (above backgrounds at 0-10, above player at 50, below UI/debug at 100)
   - **Rain preset:** emitZone covering full camera viewport width + margin (0 to 1920, y=-20), quantity scaled by intensity (base 3, max 8 per frame), speed: { min: 300, max: 500 }, angle: { min: 85, max: 95 }, lifespan: 800, scaleX: 0.5, scaleY: 3, alpha: { start: 0.4, end: 0.1 }, tint: 0x8888cc
   - **Snow preset:** emitZone (0 to 1920, y=-20), quantity scaled by intensity (base 1, max 4), speed: { min: 30, max: 80 }, angle: { min: 80, max: 100 }, lifespan: 4000, scale: { min: 0.5, max: 2 }, alpha: { start: 0.8, end: 0.2 }, tint: 0xeeeeff, gravityX with small sine oscillation via `accelerationX: Phaser.Math.Between(-20, 20)`
   - **Fog preset:** emitZone scattered across viewport bottom half, quantity: 1 per 500ms, speed: { min: 10, max: 30 }, angle: 180 (rightward drift), lifespan: 6000, scale: { start: 3, end: 6 }, alpha: { start: 0.15, end: 0 }, tint: 0xcccccc, frequency: 500
   - **Dust preset:** emitZone full viewport, quantity: 1 per 200ms, speed: { min: 5, max: 20 }, angle: random, lifespan: 3000, scale: { min: 0.5, max: 1.5 }, alpha: { start: 0.3, end: 0 }, tint: 0xddbb88, frequency: 200

   Intensity multiplies the `quantity` and `frequency` (higher intensity = more particles, lower frequency interval). Clamp intensity to [0.1, 1.0].

   All presets use `setScrollFactor(0)` so particles stay relative to camera, not world coordinates.
